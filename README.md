# 3. 分支说明

```SQL_Server```分支不再维护，因为它的语言比较繁琐，而且命令行十分难用。

```Visual Fox```原始分支更不适用。基本告别微软的数据库服务。

```master```分支为MySQL平台下的实验，以后可能跟进Oracle的平台。
