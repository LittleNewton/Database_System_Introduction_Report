# Database_System_Introduction_Report

## 1. 程序性质

- 练习性
- 学习性

## 2. 实验平台

- Ubuntu 17.10 x64
- MySQL community

实验报告在Windows 10 Pro平台完成。

## 3. 分支说明

- ```SQL_Server```分支不再维护，因为它的语言比较繁琐，而且命令行十分难用。
- ```Visual Fox```原始分支更不适用。基本告别微软的数据库服务。
- ```master```分支为MySQL平台下的实验，以后可能跟进Oracle的平台。

## 4. 实验时间跨度

2017-11-09-NOW

## 5.实验内容（重点）

- MySQL的安装连接
- 数据库的建立、表的建立、表间关系的建立
- 值的插入、更改
- 表的查询
- 存储过程与存储函数
- 视图
- 索引
- 触发器
- 访问控制与权限管理
- PHP编程调用数据库
